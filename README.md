### 模板文件说明

- 对象类型和集合类型《类型》分别填Object和List；
- 子类名默认类型去除s和List，可以通过《子类名》指定；
- 《内部类》指定是否创建与该类中；
- 《排序》目前仅用于作用上下级关系，比如<1.1>是<1>的属性；
- 《lombok》是否支持lombok；
- 排序时要从1开始；
- 排序主要要从小到大，可以跳数，但排序不能乱
- 如果是内部类，该类的属性都需要标注
- 目前仅支持单类生
